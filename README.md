# Teste de Estágio da Senac
Jogo 3D na Unity 2022.3.27f1.

## Instruções
- WASD: Movimentação.
- Espaço: Pulo.
- CuraCirculo: +1 vida.
- DanoCirculo: -1 vida.
- Vida a 0: Personagem para.
- Personagem tem um total de 100 de vida.

## Como Rodar
- Baixe a pasta `Build`.
- Execute "coliseu2.exe" no Windows.
